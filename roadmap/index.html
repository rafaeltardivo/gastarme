Where to start?

1 - First of all, you'll need to create a user
2 - Once the user was created, you can obtain a JWT Token
3 - Once authenticated, you must create a wallet
4 - With your wallet created, you'll be able to add Credit Cards to it 5 - With Credit Cards, you'll be able to make purchases
6 - Each purchase will generate a bill that should be paid to restore your credit limit

Do i need to be authenticated?

Yes, for all the actions but user creation.

How to i set my authentication?

Sending your JWT token on the Authorization header.

What are the permission levels?

User and Superuser. Users can create wallets, add credit cards, make purchases and pay bills while Superusers can list users and list and delete wallets.
